Paperless Pioneers

Dieses Projekt stellt ein einfaches Kommandozeilen-Werkzeug bereit, mit dem Text aus PDF-Dateien extrahiert werden kann.

Installation

pip install -r requirements.txt

Verwendung

python paperless.py pfad/zur/datei.pdf

Der Text des PDFs wird danach im Terminal ausgegeben.

Invoice Inc.

Der TypeScript-Code im Ordner src/ bildet eine kleine Spiele- Simulation rund um Rechnungen. Kompiliere ihn und starte die Demo mit:

npx tsc
node dist/index.js
